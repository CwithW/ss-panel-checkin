# sspanel_checkin

## Requirements
- [Golang](https://golang.org/doc/install)

## Build
1. Get source code

```
$ git clone https://github.com/CwithW/ss-panel-checkin.git
```

2. Compile the project

```
cd ss-panel-checkin
go build -o ss-panel-checkin main.go
```

## Usage

```
./ss-panel-checkin -host <host> -email <email> -passwd <password>
```
